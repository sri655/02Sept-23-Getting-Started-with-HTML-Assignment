Q2. Explain the purposeof your comments in the html and provide an example of how to use comments 
in an HTMl document?

Answer : 
In HTML, comments are used to add notes or explanations within the code that are not visible on the web page when viewed in a browser. Comments can be useful for:

Providing explanations for sections of code.
Making notes for other developers (or yourself) to understand the purpose or functionality of the code.

Example :
<!DOCTYPE html>
<html lang="en">
<head>

    <title>Document with Comments</title>
</head>
<body>
    <!-- This is the main heading of the page -->
    <h1>HTML defines the content and structure of your website</h1>
    
    

    <!-- End of the content section -->
</body>
</html>


Q3. Write and html program that includes a heading , a paragraph of text a horigental 
line and a line break  arrange these elements to create a simple layout?

Answer :
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=<device-width>, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <div class="content">
        <!-- Main heading -->
        <h1>Welcome to My Simple Layout</h1>
        
        <!-- Horizontal line under the heading -->
        <hr>
        
        <!-- Line break for spacing -->
        <br>
        
        <!-- Paragraph of text -->
        <p>This is a sample paragraph that demonstrates a simple HTML layout. The content is arranged with a heading at the top, a horizontal line underneath the heading, and a paragraph of text following that. The layout is designed to be clean and easy to read.</p>
    </div>
</body>
</html>

Q4. Write a short note on tag  and element with and example ?

Answer :
TAG :
A tag is a keyword enclosed in angle brackets that is used to mark the beginning or end of an HTML element. Tags instruct the web browser on how to interpret and display content. Tags can be:

Start tags: Indicate the beginning of an element. For example, <p> is a start tag for a paragraph.
End tags: Indicate the end of an element and are written with a forward slash before the tag name. For example, </p> is an end tag for a paragraph.


Element :
An element refers to the complete set of start tag, content, and end tag. 
It represents a piece of content and its structure on the web page. 
For example, a paragraph element in HTML consists of the start tag <p> the content and the end tag </p>.

Q5. What is doctype in Html ?

Answer :
The DOCTYPE declaration in HTML is a special instruction that tells the web browser which version of HTML the document is using. 
It helps ensure that the web browser renders the page correctly according to the specified HTML version's rules and standards.
